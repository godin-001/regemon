# ⚔️ REGEMON

> VibeCoding Bootcamp Frutero · Sesión 1

Mascota virtual estilo Pokémon/Tamagotchi construida con IA.

## Stack
- React 19 + TypeScript + Vite
- NES.css (retro pixel UI)

## Features (S1)
- 4 tipos de Regemon: Fuego, Agua, Tierra, Aire
- Stats: Hambre, Felicidad, Energía
- Ciclo de vida: Huevo → Bebé → Adulto
- Persistencia en localStorage

## Run
```bash
npm install
npm run dev
```
